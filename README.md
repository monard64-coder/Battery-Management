# Battery-Management
STM32-based battery monitoring system for a 3S LiPo battery. Uses a voltage divider and ADC to measure voltage, estimate battery percentage, and display it on a 4-digit 7-segment display. Includes green, yellow, and red LEDs for real-time battery status indication. Demo video included.
